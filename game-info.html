Stewardship Life

David Strege’s vision: steward time, talents, and treasures. Plans come first; events test them. The home screen is a version lab so testers can compare approaches.

  • V1 Classroom: wizard → lifetime projection → teacher debrief.
  • V2 Ongoing: year-by-year baseline (25→85).
  • V3 Vision+: David’s playtest wins — Joy label, weekly hours meter, job change, denser pop-ups, Home confirm + autosave, Feedback + optional Coach.
  • V4 AI Wizard: same controls; small-model coach interprets numbers and suggests tweaks (Railway + API key for live LLM; otherwise local rules).
  • Score: Wealth × (1 + Health% + Joy%). Money alone is not the win.
  • Feedback: in-header button captures note + game state + screenshot so testers can collab without email threads.

Product map: docs/directions.md. David learning to edit via AI: open this repo in Cursor and ask the agent to change the game — that’s the “edit through AI” path (not a separate in-game editor yet).
